# Digital Transformation Strategy: Vaccination Management (An Bien Commune)

## 📌 Project Overview
Strategic proposal to digitalize the paper-based vaccination management for UBND Xã An Biên (An Giang). 
**Goal:** Transitioning from manual records to a Cloud-based ecosystem (CIVRM).

## 📊 Strategic Analysis & Maturity Assessment
We evaluated the digital readiness using the **Digital Maturity Model (DMM)** and environmental factors.
* **Current State:** Level 1 (Initiated)
* **Frameworks used:** SWOT, PESTLE, Digital Maturity Model (DMM).

![Strategic Analysis](image\image_SWOT.png)
*Figure: Strategic analysis and Digital Maturity assessment*

## ⚙️ Process Optimization (BPMN 2.0)
Redesigning the manual workflow into an automated "TO-BE" model to improve efficiency.
* **Key Impact:** Estimated 60% reduction in processing time.
